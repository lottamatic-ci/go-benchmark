# go-benchmark

This repository is a small demonstration project written in Go.  
Its purpose is to showcase basic Go project structure, modular design, and clean code practices.

## What is demonstrates

The program reads user input from the command line, counting the number of words (tokenization), vowels. The program demonstrates packet organization (e.g. differentiation between internal and pkg packets) and basic string processing. 

## 🚀 How to Run

Just run with
```bash
go run main.go
